# ✝️ Nicho Cristão

Projeto completo para nicho cristão: Landing Pages, SaaS e Automação de Conteúdo.

## 📁 Estrutura do Projeto

### Landing Pages
Templates e componentes para páginas de captura e vendas
- Páginas de captura (lead magnet)
- Páginas de vendas (produtos digitais)
- Páginas de eventos (cultos, conferências)
- Thank you pages

### SaaS
Aplicação completa (backend + frontend + database)
- Gestão de membros
- Plataforma de cursos
- Sistema de doações
- Dashboard administrativo

### Workflows (n8n)
Automações para produção e publicação de conteúdo
- **video-automation/** - Geração automática de vídeos
- **social-posting/** - Publicação automática nas redes
- **content-generation/** - Criação de conteúdo com IA

### Social Media
Scripts e templates para redes sociais
- Templates de vídeos
- Thumbnails
- Legendas automatizadas
- Hashtags otimizadas

## 🎥 Automação de Vídeos

Sistema automatizado que:
1. Gera ideias de conteúdo cristão trending
2. Cria roteiros baseados na Bíblia
3. Gera thumbnails
4. Produz legendas
5. Publica automaticamente em múltiplas plataformas

Plataformas:
- YouTube
- Instagram Reels
- TikTok
- Facebook

## 🛠️ Stack Tecnológica

### Landing Pages
- HTML/CSS/JS ou React
- Tailwind CSS
- Vercel (deploy)

### SaaS
- Frontend: React + Vite
- Backend: Node.js / Python
- Database: Supabase
- Auth: Supabase Auth

### Automação
- n8n (workflows)
- Claude API (geração de conteúdo)
- YouTube API
- Instagram API
- TikTok API

### AIOS
- Agentes IA para desenvolvimento
- 7 Expert Skills do n8n
