# Músicas de Fundo — Royalty-Free

Coloque aqui arquivos `.mp3` ou `.wav` para uso como trilha de fundo nos vídeos.

## Fontes recomendadas (gratuitas, uso comercial)

| Fonte | URL | Obs |
|-------|-----|-----|
| Pixabay Music | https://pixabay.com/music/ | 100% livre, sem atribuição |
| YouTube Audio Library | https://www.youtube.com/audiolibrary | Grátis para canais YT |
| FreePD | https://freepd.com/ | Domínio público |
| ccMixter | http://ccmixter.org/ | Creative Commons |

## Termos de busca sugeridos

- `inspirational piano`
- `peaceful orchestral`
- `holy ambient`
- `christian meditation`
- `divine strings`
- `spiritual background`
- `heavenly music`

## Nomes de arquivo sugeridos

```
peaceful_hymn_01.mp3
inspirational_strings_02.mp3
celestial_piano_03.mp3
divine_orchestral_04.mp3
holy_ambient_05.mp3
```

O script `assemble-video.sh` selecionará uma trilha aleatória a cada execução.
